POST REQUEST IS MADE TO /meetings using POSTMAN

For creating meeting:

http://localhost:8080/meetings

For Accessing all meeting:

A sample post request is as follows(Id for each meeting is randomly generated!!)

{ "title":"cultural meet", "participants":[{ "name":"neeraj", "email": "neeraj@gmail.com", "rsvp": "MayBe" }, { "name":"shubham", "email": "shubham@gmail.com", "rsvp": "Yes" }], "startTime":"12:34:56, TuesDay", "endTime":"13:34:56, TuesDay", "CreationTimeStamp": "12:34:58" }

http://localhost:8080/meetings

For Accessing specific meeting with given id:


http://localhost:8080/meetings/{id}

For Accessing specific meeting with given email:


http://localhost:8080/meetingd?email=email

For Accessing all meetings between start time and between end time

http://localhost:8080/meetingd?email=email
